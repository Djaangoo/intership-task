# Intreship Task

## Installation

install [node ](https://nodejs.org/en/download/) (16.x.x)

install yarn

```bash
npm install --global pnpm
```

install packages

```bash
pnpm install
```

## Scripts

Developer build with HMR

```bash
pnpm dev
```

Lint

```bash
pnpm lint
```

Format (resolve fixable bugs)

```bash
pnpm format
```

Prod Build

```bash
pnpm build
```

Preview Build

```bash
pnpm preview
```
