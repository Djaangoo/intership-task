
# Intreship Task

  

## Installation

install [node ](https://nodejs.org/en/download/) (16.x.x)
 
 install yarn
```bash
npm install --global yarn
```
install packages
```bash
yarn install
```

## Scripts

Developer build with HMR
```bash
yarn dev
```
Lint
```bash
yarn lint
```
Format (resolve fixable bugs)
```bash
yarn format
```
Prod Build
```bash
yarn build
```
Preview Build
```bash
yarn preview
```
## Tutorial

 - Install project
 - run:  ```yarn dev ```
 - open [localhost](http://localhost:3000/)
 - rest of task will be in the app 😁😁
